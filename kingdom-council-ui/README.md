# AI Kingdom Council

## Project Description

The **AI Kingdom Council** is a multi-agent AI system inspired by ancient Himalayan kingdoms, where diverse perspectives from specialized advisors unite to guide strategic decisions.

## Project Info

A collaborative decision-making platform featuring:

- Multi-agent council with specialized advisors (The King, Senapati, Guru, Mantri)
- Real-time deliberations and Royal Decrees
- Kingdom dashboard and chronicles
- Himalayan-themed UI/UX

## Technologies Used

This project is built with:

- **Vite** - Fast build tool and dev server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - Beautiful component library
- **Tailwind CSS** - Utility-first CSS framework
- **GSAP** - Advanced animations
- **React Router** - Navigation

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd kingdom-council-ui

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```
